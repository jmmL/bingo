# 2026 Predictions Bingo

A fun, randomly generated bingo grid for 2026 predictions.

## Features
- **5x5 Grid**: Randomly generated from a curated list of predictions.
- **Categorized Logic**: Ensures a balanced mix of Politics, Tech, Economics, Science, and Culture.
- **Interactive**: Click squares to stamp them with a realistic ink mark.
- **Export**: Download your unique bingo card as a PNG image.
- **Mobile Friendly**: Responsive design that works on phones and desktops.

## Deployment
This site is designed to be hosted directly on **GitHub Pages**.

1. Go to the repository **Settings**.
2. Scroll down to the **Pages** section (or click "Pages" in the sidebar).
3. Under **Source**, select `Deploy from a branch`.
4. Under **Branch**, select `main` (or the branch this code is in) and `/ (root)`.
5. Click **Save**.
6. Wait a moment for the URL to appear!
